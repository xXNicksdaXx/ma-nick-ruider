\chapter{State Of The Art}
\label{chapter:state_of_the_art}
% ==========================================================

This chapter surveys the scientific and technical landscape relevant to this thesis across three complementary domains.
They cover modeling of feature trees, Behavioral Programming comprehension, and implementation of graphical editor platforms.

The following collection of works represents the most relevant contributions in these overlapping areas.

The chapter first reviews the Universal Variability Language and the ecosystem of tools that enable formal architectural specification through feature modeling.
It then examines Behavioral Programming and the specialized tools that have emerged to help developers understand the dynamic behavior of \gls{bp} systems.
Finally, the Graphical Language Server Platform is discussed as a modern, extensible framework for building domain-specific diagram editors.
